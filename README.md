**YouTube Music Desktop** — это удобное и легковесное десктопное приложение для прослушивания музыки через сервис YouTube Music. Больше не нужно держать открытой лишнюю вкладку в браузере: управляйте треками, плейлистами и подкастами прямо из отдельного независимого окна с глубокой интеграцией в операционную систему.

---

## ✨ Особенности и возможности

* **🎵 Полноценный плеер:** Доступ ко всей вашей медиатеке YouTube Music, персональным миксам, рекомендациям и новинкам.
* **⌨️ Поддержка медиа-клавиш:** Полноценное управление воспроизведением (плей/пауза, следующий/предыдущий трек) с помощью мультимедийных кнопок клавиатуры или гарнитуры.
* **💻 Нативная интеграция:** Отображение текущего трека в системном трее и поддержка всплывающих уведомлений при смене композиции.
* **🎨 Чистый интерфейс:** Минималистичный дизайн, не отвлекающий от музыки, с возможностью сворачивания в фоновый режим.
* **🚀 Экономия ресурсов:** Потребляет значительно меньше оперативной памяти, чем полноценный браузер (Chrome, Firefox и др.).

---

## 🚀 Установка и запуск

### Стабильная версия (Релизы)
1. Перейдите в раздел **Releases**.
2. Установите и запустите приложение.

### Сборка из исходного кода
Если вы хотите собрать проект самостоятельно, выполните следующие команды в терминале:
dotnet publish -c Release -r win-x64 --self-contained true -p:PublishSingleFile=true
